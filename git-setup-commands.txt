# ContractGuard Git Setup Commands
# Run these commands in order after installing Git

# 1. Configure Git (replace with your info)
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# 2. Initialize repository
git init

# 3. Add all files
git add .

# 4. Create initial commit
git commit -m "Initial commit: ContractGuard AI-powered contract analysis platform

Features:
- Complete React frontend with Tailwind CSS
- Node.js backend with Express and MongoDB
- AI-powered contract analysis with OpenAI integration
- User authentication and Pro subscription system
- PDF upload and processing capabilities
- Risk detection and scoring algorithms
- Responsive dashboard with analytics
- Comprehensive error handling and security features"

# 5. Add GitHub remote (replace 'yourusername' with your GitHub username)
git remote add origin https://github.com/yourusername/contractguard.git

# 6. Push to GitHub
git branch -M main
git push -u origin main
